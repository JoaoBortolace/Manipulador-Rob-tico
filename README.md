# Placa de Controle

Placa desenvolvida para controlar os motores CC de cada junta, ela é baseado no microcontrolador STM32G4, responsável por gerar os sinais PWM de controle para o acionamento dos motores pela pontes-H, enquanto fecha a malha atráves da leitura de corrente e encoders de cada motor. A alimentação é feita através do USB-PD.
